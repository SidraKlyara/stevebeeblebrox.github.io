What is Mixin?

Mixin is a modding API that allows developers to easily hook into existing vanilla code. Unlike other similar APIs, Mixin does not need to be added to the build.gradle file or shipped with each mod. Developers simply create files that define changes for Mixin to apply. Each of these files is only 10 lines. Developers can even hook into each other's code allowing for better compatibility. For security, developers can set certain parts of their code to be protected from changes. Mixin quickly and effectively applies its changes. There is no separate installer needed. Users only have to run the mod once then restart the game to set some things up.
